## MCP
MCP (Model Context Protocol) was officially proposed and open-sourced by the AI company Anthropic on `November 25, 2024`. This protocol aims to provide a standardized interface for large language models (such as Claude3.7 sonnect) to connect and interact with external data sources and tools. Because it can call various external tools, it is considered a key node in driving the `AI Agent` revolution, helping the currently popular `AI Agent` models achieve the following functions:
1. Dynamically access files, APIs, and databases
2. Read and invoke tools or plugins
3. Interact with operating systems or browsers

## The Impact of MCP on the Current AI Landscape
### 1. Full-process Automated AI Agents Represented by Manus
ManusAI is a general-purpose autonomous AI Agent launched by the Chinese startup Monica on `March 6, 2025`. It aims to achieve full-process automation from user intent to actual execution. Users only need to provide the problem and the materials required by the AI, and it will automatically create virtual machines and call various tools in the background to complete the task.

![manus](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/manus%E4%BD%BF%E7%94%A8.png)
### 2. Workflow AI Agents Represented by Coze
Coze is a lightweight AI Agent building platform launched by ByteDance. It breaks down tasks into workflow nodes through a "module drag-and-drop + dialogue-driven" approach, allowing users to create AI Agents that can automatically execute complex tasks without programming.
![coze](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/coze.png)
### 3. IDE AI Agents Represented by Cline
Cline is a new generation development assistant AI Agent launched by `Moonshot AI`. It is embedded in mainstream IDEs and supports advanced functions such as code refactoring driven by natural language instructions, cross-file debugging, and unit test generation. The underlying architecture of Cline also adopts the MCP protocol, enabling it to call external agents when needed to assist in completing complex tasks, such as invoking a browser or performance analysis tools, achieving parallel collaboration of multiple agents in the development process.
![cline](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/cline.png)
### 4. Open-source Full-process AI Agents Represented by OpenManus
OpenManus is an open-source project based on the MCP protocol, driven by community developers. Its goal is to provide a full-process AI Agent system that can run without relying on closed-source services. Similar to ManusAI, OpenManus also supports users starting from natural language input, with the Agent automatically planning tasks, creating execution environments, calling toolchains, and achieving goals. The difference is that OpenManus is fully open-source, allowing users to freely customize their agent's behavior, plugin interfaces, and tool invocation logic, greatly promoting the community application and ecosystem development of the MCP protocol.
![open manus](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/openManus.png)

## ProverModel
On April 30, 2025, DeepSeek officially released its new generation mathematical theorem proving model, DeepSeek-Prover-V2, on the AI open-source community HuggingFace. This model is the world's first formal mathematical large model. Its 671B model possesses the most rational mathematical reasoning capabilities currently available. ## The Impact of ProverModel on the Current AI Landscape
Unlike current hybrid reasoning models, the data generated by mathematical theorem proving models can serve as excellent rational training data for training other models, accelerating the transition from irrational to rational models, thereby moving more quickly towards AGI.

## Summary
MCP and ProverModel are important innovations in the current AI landscape, and they hold significant meaning for the development of AI.
