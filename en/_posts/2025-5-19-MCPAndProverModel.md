## MCP
MCP (Model Context Protocol) was officially proposed and open-sourced by the AI company Anthropic on `2024 年 11 月 25 日`. This protocol aims to provide a standardized interface for large language models (such as Claude3.7 sonnect) to connect and interact with external data sources and tools. Because it can invoke a variety of external tools, it is considered a key node in driving the `AI Agent` revolution, helping the currently popular `AI Agent` models achieve the following functions:
1. Dynamically access files, APIs, and databases
2. Read and invoke tools or plugins
3. Interact with operating systems or browsers

## MCP's Impact on the Current AI Landscape
### 1. Full-process Automated AI Agents Represented by Manus
ManusAI is a general-purpose autonomous AI Agent launched by the Chinese startup Monica on `2025 年 3 月 6 日`. It aims to achieve full-process automation from user intent to actual execution. Users only need to provide the problem and the materials required by the AI, and it will automatically create virtual machines and invoke various tools in the background to complete the task.

![manus](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/manus%E4%BD%BF%E7%94%A8.png)
### 2. Workflow AI Agents Represented by Coze
Coze is a lightweight AI Agent building platform launched by ByteDance. It decomposes tasks into workflow nodes through a "module drag-and-drop + dialogue-driven" approach, allowing users to create AI Agents that can automatically execute complex tasks without programming.
![coze](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/coze.png)
### 3. IDE AI Agents Represented by Cline
Cline is a new generation development-assistant AI Agent launched by `Moonshot AI`. It is embedded and integrated into mainstream IDEs, supporting advanced functions such as code refactoring driven by natural language instructions, cross-file debugging, and unit test generation. Cline's underlying architecture also adopts the MCP protocol, enabling it to invoke external agents when needed to assist in completing complex tasks, such as calling a browser or performance analysis tools, achieving parallel collaboration of multiple agents in the development process.
![cline](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/cline.png)
### 4. Open-source Full-process AI Agents Represented by OpenManus
OpenManus is an open-source project based on the MCP protocol, driven by community developers. Its goal is to provide a full-process AI Agent system that can run without relying on closed-source services. Similar to ManusAI, OpenManus also supports users starting from natural language input, with the Agent automatically planning tasks, creating execution environments, invoking tool chains, and completing objectives. The difference is that OpenManus is fully open-source, allowing users to freely customize its agent behavior, plugin interfaces, and tool invocation logic, greatly promoting the community application and ecosystem development of the MCP protocol.
![open manus](https://cdn.jsdelivr.net/gh/byolio/tc3@main/img/openManus.png)

## ProverModel
On April 30, 2025, DeepSeek officially released its new generation mathematical theorem proving model, DeepSeek-Prover-V2, on the AI open-source community HuggingFace. This model is the world's first formal mathematics large model. Its 671B model possesses the most rational mathematical reasoning ability currently. ## ProverModel's Impact on the Current AI Landscape
Unlike current hybrid reasoning models, the data generated by mathematical theorem proving models can serve as excellent rational training data for training other models, accelerating the transition of non-rational models towards rationality, thereby achieving a faster path to AGI.

## Summary
MCP and ProverModel are important innovations in the current AI field, and they hold significant importance for the development of AI.
