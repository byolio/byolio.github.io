## What is PyTorch
PyTorch is an open-source deep learning framework developed by `Facebook`'s AI Research lab (FAIR). It is known for its dynamic computation graphs, intuitive API, and powerful GPU acceleration capabilities. PyTorch is primarily used in fields such as computer vision and natural language processing, enabling researchers and developers to quickly build and train neural network models.
## Why Choose PyTorch
1. Dynamic Computation Graphs: PyTorch uses a dynamic computation graph mechanism, allowing network structures to be changed at runtime, facilitating debugging and development.
2. Ease of Use: PyTorch's API design is intuitive and close to Python's coding style, lowering the learning curve.
3. Strong Community Support: It has an active community and numerous third-party libraries, making the development process more convenient.
4. High Extensibility: Supports custom layers and operations, making development easier.
## PyTorch Installation (using Windows as an example)
1. Install CUDA (GPU version of PyTorch requires CUDA)
Open the terminal and enter:
```bash
nvidia-smi
```
Check the driver version. The [official CUDA documentation](https://docs.nvidia.com/deploy/cuda-compatibility/) lists the minimum driver version requirements for each toolkit version. For example, `CUDA 12.7` typically requires driver version `525.85.12` or higher.
![cudaVision](https://cdn.jsdelivr.net/gh/byolio/newtc@main/img/cudaVision.png)
Download the CUDA driver [link](https://developer.nvidia.com/cuda-downloads?target_os=Windows&target_arch=x86_64&target_version=11&target_type=exe_local) (I have already selected it, just download).
![CUDA](https://cdn.jsdelivr.net/gh/byolio/newtc@main/img/CUDA.png) Restart your computer after installation.
After restarting, open the terminal and enter:
```bash
nvcc -V
``` 
If CUDA-related information is displayed, the installation was successful.
2. Install PyTorch
Open the [PyTorch official download page](https://pytorch.org/get-started/locally/)
![pytorchDownload](https://cdn.jsdelivr.net/gh/byolio/newtc@main/img/pytorchDownload.png) Select the corresponding command and copy it to the terminal to download (it is not recommended to download to the base environment).
## Installation of Common PyTorch Tools
I use conda for installation here. [Method to install conda](https://byolio.top/2024/10/03/conda/).
1. Jupyter (a brief introduction, no advanced Jupyter content will be written)
Conda comes with Jupyter. I will mainly explain how to create a virtual environment with PyTorch installed and import that virtual environment into Jupyter. \
Use the following code (note to use your own Python version, copy `python -V` to the terminal to check your Python version):
```bash
conda create -n pytorch python=3.10
conda activate pytorch
```
Then install PyTorch in this environment using the previously used PyTorch installation command. Restart the terminal after installation.
Create the corresponding IPython kernel:
```bash
python -m ipykernel install --user --name=pytorch --display-name "Pytorch_env"
```
Start Jupyter:
```bash
jupyter notebook
```
You will then see the newly added Jupyter kernel.
2. TensorBoard (optional)
Switch to the environment with `PyTorch` and install:
```bash
conda install tensorboard
```
Start TensorBoard:
```bash
tensorboard --logdir=logs
```
`logs` is the folder for saving logs, please replace it accordingly.
3. PyCharm
A basic IDE tool, no further explanation needed. Just [download and install](https://www.jetbrains.com/pycharm/download/?section=windows). Advanced content will be written later. \
Note that the interpreter must be in an environment with `PyTorch`.
You can test it using `import torch`.

## FAQ
1. If the conda command cannot be used
Check that the correct environment variables are set, then enter in the terminal:
```bash
conda init
```
2. How to test if GPU is available
```python
import torch
print(torch.cuda.is_available())
```
If it returns True, it means the GPU is available.
3. How to test GPU computing capability
```python
import torch
print(torch.cuda.get_device_capability())
```
