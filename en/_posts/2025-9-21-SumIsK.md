## Introduction
Subarray sum equals k is a classic problem. Its optimal solution is O(n), using prefix sum + hash table.

## Implementation Code
```c++
#include<unordered_map>
class Solution {
public:
    unordered_map<int, int> map;   // values in nums array, indices of nums
    int subarraySum(vector<int>& nums, int k) {
        int n = nums.size();
        int ans = 0;
        int sum = 0;
        map[0] = 1;   // initialize the count of prefix sum 0 to 1
        for(int i = 0; i < n; ++i){
            sum += nums[i];
            if(map.count(sum - k) > 0){
                ans += map[sum - k];
            }
            map[sum]++;
        }
        return ans;
    }
};
```
## Explanation
1. Use a hash table `map` to store the occurrence count of prefix sums. Initialize `map[0] = 1`, indicating that the prefix sum of 0 occurs once (when no numbers have been accumulated).
2. Since the sum value is recorded via the map and has a forward-propagation property, the `vector<int> sums` can be optimized to a single sum variable, reducing space complexity.
3. Use the hashmap to record the occurrence count of sum values. When `sum - k` appears in the hashmap, it indicates that there exists a previous subarray whose cumulative sum is `k`.
4. During array traversal, compute the current prefix sum `sum` each time, and check if `sum - k` exists in `map`. If it does, it means there is a subarray with cumulative sum `k`; add the corresponding count to the answer, and then record the current prefix sum `sum` into `map`.
