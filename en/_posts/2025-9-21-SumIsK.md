## Introduction
Subarray sum equals k is a classic problem. Its optimal solution is O(n), using prefix sum + hash table.

## Implementation Code
```c++
#include<unordered_map>
class Solution {
public:
    unordered_map<int, int> map;   // nums array values, nums indices
    int subarraySum(vector<int>& nums, int k) {
        int n = nums.size();
        int ans = 0;
        int sum = 0;
        map[0] = 1;   // Initialize the count of prefix sum 0 to 1
        for(int i = 0; i < n; ++i){
            sum += nums[i];
            if(map.count(sum - k) > 0){
                ans += map[sum - k];
            }
            map[sum]++;
        }
        return ans;
    }
};
```
## Explanation of Logic
1. Use a hash table `map` to store the frequency of prefix sums. Initialize `map[0] = 1`, indicating that the prefix sum of 0 appears once (when no numbers have been accumulated).
2. Since the sum value is recorded via the map and has the property of moving forward, we can optimize the `vector<int> sums` to a single sum value, reducing space complexity.
3. Use a hashmap to record the frequency of sum values. When `sum - k` appears in the hashmap, it indicates that there exists a previous subarray whose cumulative sum is k.
4. While traversing the array, each time calculate the current prefix sum `sum`, and check if `sum - k` exists in `map`. If it exists, it means there is a subarray with cumulative sum `k`, add the corresponding frequency to the answer, and record the current prefix sum `sum` into `map`.
